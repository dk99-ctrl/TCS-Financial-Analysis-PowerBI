TCS Financial Analysis and Revenue Forecasting Dashboard
Project Overview

This project presents a detailed financial analysis of Tata Consultancy Services (TCS) covering FY2019 to FY2025, along with a revenue forecast for FY2026.

The objective was to evaluate the company’s growth performance, profitability trends, capital structure, and financial stability using Power BI and DAX-based financial modeling.

The dashboard is structured into four key analytical sections:

Executive Overview

Profitability Analysis

Financial Health

Revenue Forecast 2026
